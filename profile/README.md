## Solving PocketBase Setup Problems

Auth rules block users: recheck pocketbase auth collection rules with a normal account, then compare behavior against pocketbase documentation examples.  
Realtime events do not appear: confirm the client subscribes to the correct collection and that pocketbase realtime is reachable through the proxy.  
Deployment feels unclear: write a small pocketbase deployment runbook covering backups, upgrades, logs, and rollback before production traffic.
